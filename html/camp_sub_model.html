CAMP: Sub-Model (general)

A sub-model is used during solving to calculate parameters based on the current model state for use by reactions.

The available sub-models are:

  • PDFiTE Activity Coefficients
  • UNIFAC Activity Coefficients
  • ZSR Aerosol Water

The general input format for a sub-model can be found here.

CAMP: Chemical Mechanism

A mechanism in the camp-chem module is a set of reactions that occur in the gas-phase or within one of several aerosol phases or across an interface between two phases (gas or aerosol). One or several mechanisms may be included in a camp-chem model run.

Every mechanism in a camp-chem run will have access to the same set of chemical species and aerosol phases, so phase and species names must be consistent across all concurrently loaded mechanisms. The division of reactions into distinct mechanisms permits a host model to specificy which mechanisms should be solved during a call to camp_camp_core::camp_core_t::solve().

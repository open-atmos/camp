◆ calculate_A()

double calculate_A ( double T,
double M,
double n )

Calculates the Troe-like parameter A(T, [M], n)

k0 = 2e-22 e^n kinf = 0.43 * (T/298)^-8 A = k0 [M] / ( 1 + k0 [M] / kinf ) * 0.41^( 1 / ( 1 + [ log10( k0 [M] / kinf ) ]^2 ) )

Parameters
Ttemperature [K]
Mair density [#/cc]
nnumber of heavy atoms in RO2 species

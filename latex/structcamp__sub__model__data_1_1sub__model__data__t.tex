\doxysubsection{Detailed Description}
Abstract sub-\/model data type. 

Time-\/invariant data required by a sub-\/model to perform calculations during solving. Derived types extending \doxylink{structcamp__sub__model__data_1_1sub__model__data__t}{sub\+\_\+model\+\_\+data\+\_\+t} should be related to specific models or parameterizations. Sub models will have access to the current model state during solving, but will not be able to modify state variables or contribute to derivative arrays directly (this must be done by reactions).

See \doxylink{camp_sub_model}{Sub Models} for details.

\doxysection{CAMP\+: Reactions (general)}
\hypertarget{camp_rxn}{}\label{camp_rxn}\index{CAMP: Reactions (general)@{CAMP: Reactions (general)}}
A reaction represents a transformation of the model state due to a physical or chemical process that occurs within a phase (gas or \doxylink{camp_aero_phase}{aerosol}) or across the interface between two phases. In the \doxysectlink{index}{camp-\/chem}{0} model, reactions are grouped into \doxylink{camp_mechanism}{mechanisms}, which are solved over time-\/steps specified by the host model.

The primary function of a reaction in the \doxysectlink{index}{camp-\/chem}{0} model is to provide the solver with contributions to the time derivative and Jacobian matrix for \doxylink{camp_species}{chemical species} concentrations based on the current model state described in a {\ttfamily \doxylink{structcamp__camp__state_1_1camp__state__t}{camp\+\_\+camp\+\_\+state\+::camp\+\_\+state\+\_\+t}} object.

Specific reaction types extend the abstract {\ttfamily \doxylink{structcamp__rxn__data_1_1rxn__data__t}{camp\+\_\+rxn\+\_\+data\+::rxn\+\_\+data\+\_\+t}} type and generally accept a set of reactants and products whose names correspond to \doxylink{camp_species}{chemical species} names, as well as a set of reaction parameters needed to describe a particular reaction. During initialization, a reaction will have access to its set of parameters as well as the parameters of any \doxylink{camp_species}{species} and \doxylink{camp_aero_rep}{aerosol phase} in the \doxysectlink{index}{camp-\/chem}{0} model, however this information will not be available during a model run. The information required by the reaction instance to calculate its contribution to the time derivatve and Jacobian matrix must therefore be packed into the condensed data arrays of the {\ttfamily \doxylink{structcamp__rxn__data_1_1rxn__data__t}{camp\+\_\+rxn\+\_\+data\+::rxn\+\_\+data\+\_\+t}} object during intialization.

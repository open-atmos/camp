\doxysubsection{Detailed Description}
Property link data. 

An element of a property data set. Property values can be of any primitive type or be a pointer to a sub-\/set of property data. The \doxylink{structcamp__property_1_1property__link__t}{property\+\_\+link\+\_\+t} object is for internal use in the \doxylink{namespacecamp__property}{camp\+\_\+property} module. All interactions with property data should be made using \doxylink{structcamp__property_1_1property__t}{property\+\_\+t} objects.

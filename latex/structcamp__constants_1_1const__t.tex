\doxysubsection{Detailed Description}
Physical constants. 

These are all absolute constants. Environment parameters that in principle could change should go in scenario.\+f90.

To access a constant in a subroutine you should {\ttfamily use \mbox{\hyperlink{namespacecamp__constants}{camp\+\_\+constants}}} and then the constant value is accessed with {\ttfamily const\%pi} or similar. Note that the type is called {\ttfamily \mbox{\hyperlink{structcamp__constants_1_1const__t}{const\+\_\+t}}} (with a trailing \+\_\+t) but the single saved variable to access them is just called {\ttfamily const}.

\doxysectlink{index}{CAMP}{0} uses two types of input files\+:


\begin{DoxyItemize}
\item \doxylink{input_format_camp_file_list}{File List} A {\ttfamily json} file containing a list of \doxysectlink{index}{CAMP}{0} configuration file names.
\item \doxylink{input_format_camp_config}{Configuration File} One or more {\ttfamily json} files containing all the \doxysectlink{index}{CAMP}{0} configuration data.
\end{DoxyItemize}

To initialize \doxysectlink{index}{CAMP}{0} , the path to the \doxylink{input_format_camp_file_list}{file list} must be passed to the \doxylink{structcamp__camp__core_1_1camp__core__t}{camp\+\_\+camp\+\_\+core\+::camp\+\_\+core\+\_\+t} constructor.

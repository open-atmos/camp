\doxysubsection{Detailed Description}
Property data. 

A set of physical properties, sub-\/model parameters and similar constants related to a chemical species, reaction, or other data object. The {\ttfamily \doxylink{structcamp__property_1_1property__t}{property\+\_\+t}} type can be used to build a set of data with a {\ttfamily json} -\/like structure.

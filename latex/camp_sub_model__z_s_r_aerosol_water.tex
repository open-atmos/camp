\doxysubsubsection{CAMP\+: ZSR Aerosol Water}
\hypertarget{camp_sub_model__z_s_r_aerosol_water}{}\label{camp_sub_model__z_s_r_aerosol_water}\index{CAMP: ZSR Aerosol Water@{CAMP: ZSR Aerosol Water}}
ZSR aerosol water calculates equilibrium aerosol water content based on the Zdanovski-\/\+Stokes-\/\+Robinson mixing rule {\bfseries [Jacobson1996]} in the following generalized format\+:

\[
   W = \sum\limits_{i=0}^{n}\frac{1000 M_i}{MW_i m_{i}(a_w)}
 \]

where $M$ is the concentration of binary electrolyte $i$ ( $\mbox{$\mu$g}\,\mbox{m}^{-3}$) with molecular weight $MW_i$ ( $\mbox{kg}\,\mbox{mol}^{-1}$) and molality $m_{i}$ at a given water activity $a_w$ (RH; 0--1) contributing to the total aerosol water content $W$ ( $\mbox{$\mu$g}\,\mbox{m}^{-3}$).

 The key-\/value pair {\bfseries{aerosol}} {\bfseries{phase}} is required to specify the aerosol phase for which to calculate water content. Key-\/value pairs {\bfseries{gas-\/phase}} {\bfseries{water}} and {\bfseries{aerosol-\/phase}} {\bfseries{water}} must also be present and specify the names for the water species in each phase. The final required key-\/value pair is {\bfseries{ion}} {\bfseries{pairs}} which should contain a set of key-\/value pairs where the key of each member of the set is the name of a binary electrolyte and the contents contain parameters required to estimate the contribution of the this electrolyte to total aerosol water. The name of the electrolyte may or may not refer to an actual aerosol-\/phase species.

Each binary electrolyte must include a {\bfseries{type}} that refers to a method of calculating ion-\/pair contributions to aerosol water. Valid values for {\bfseries{type}} are {\bfseries{JACOBSON}} and {\bfseries{EQSAM}}. These are described next.

Aerosol water from ion pairs with type {\bfseries{JACOBSON}} use equations (28) and (29) in Jacobson et al. (1996) {\bfseries [Jacobson1996]} where experimentally determined binary solution molalities are fit to a polynomial as\+:

\[
   \sqrt{m_{i}(a_w)} = Y_0 + Y_1 a_w + Y_2 a_w^2 + Y_3 a_w^3 + ...,
 \]

where $Y_j$ are the fitting parameters. Thus, $m_i(a_w)$ is calculated at each time step, assuming constant $a_w$. These values must be included in a key-\/value pair {\bfseries{Y\+\_\+j}} whose value is an array with the $Y_j$ parameters. The size of the array corresponds to the order of the polynomial equation, which must be greater than 1. The key-\/value pair {\bfseries{low}} {\bfseries{RH}} is required to specify the lowest RH (0--1) for which this fit is valid. This value for RH will be used for all lower RH in calculations of $m_i(a_w)$ as per Jacobson et al. (1996) {\bfseries [Jacobson1996]}.

The key-\/value pair {\bfseries{ions}} must contain the set of ions this binary electrolyte includes. Each species must correspond to a species present in {\bfseries{aerosol}} {\bfseries{phase}} and have a {\bfseries{charge}} parameter that specifies their charge (uncharged species are not permitted in this set) and a {\bfseries{molecular}} {\bfseries{weight}} {\bfseries{}}\mbox{[}{\bfseries{kg}} {\bfseries{mol-\/1}}\mbox{]} ( $\mbox{kg}\,\mbox{mol}^{-1}$) property. Ions without a {\bfseries{qty}} specified are assumed to appear once in the binary electrolyte. The total molecular weight for the binary electroly $MW_i$ is calculated as a sum of its ionic components, and the ion species concentrations are used to determine the $M_i$ during integration. The aerosol-\/phase water species to be calculated using the ZSR method must have the key-\/value pair {\bfseries{tracer}} {\bfseries{type}} with value "{}\+CONSTANT"{}.

 Aerosol water from ion pairs with type {\bfseries{EQSAM}} use the parameterization of Metzger et al. (2002) {\bfseries [Metzger2002]} for aerosol water content\+:

\[
   \sqrt{m_{i}(a_w)} = (NW_i MW_{H2O}/MW_i (1/a_w-1))^{ZW_i}
 \]

where $NW_i$ and $ZW_i$ are fitting parameters {\bfseries [Metzger2002]}, and must be provided in key-\/value pairs {\bfseries{NW}} and {\bfseries{ZW}}, along with the binary electrolyte molecular weight {\bfseries{MW}} ( $\mbox{kg}\,\mbox{mol}^{-1}$). The key-\/value pair {\bfseries{ions}} must contain a set of ions that can be summed to calculate $M_i$ at runtime.
